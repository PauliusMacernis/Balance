## Running

To start a web server for the application, run:

    lein run -p 3000
    
To run migrations:

    lein run migrate
    
To start the application from the REPL using `start` function defined in the `user` namespace:

    lein repl
    
To run test, once:
    
    lein test
    
To run test, constantly:
    
    lein test-refresh
